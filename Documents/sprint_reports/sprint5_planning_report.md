## Sprint 5: UI Localization Planning

**Goal:** Enable full multilingual support and non-Latin GUI scalability.
**Duration:** 2 weeks.

---

### **Key Objectives**
* **UI Prep:** Externalize all static text and implement selectors for language, dates, and tables.
* **Implementation:** Support **two non-Latin languages** with dynamic LTR/RTL layout updates.
* **Backlog:** Add localization stories, acceptance criteria, and estimates to Trello/Jira.
* **Docs:** Update `README.md` with setup instructions and add new UML diagrams.

---

### **Deliverables**
* Functional localized UI with 2 non-Latin options.
* Documentation of the localization framework/library used.
* Sprint 6 Plan (Database Localization focus) and Sprint 5 Review Summary.
